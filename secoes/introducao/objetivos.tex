Nesta seção serão apresentados os objetivos do trabalho, de forma a esclarecer as metas estabelecidas durante a elaboração.

\subsection{Objetivo Geral}

Este trabalho busca propor uma plataforma multiagente para suportar o desenvolvimento de agentes de gerenciamento de ambientes de CN, visando uma melhor eficiência energética em ambientes de CN.

\subsection{Objetivos Específicos}

\begin{itemize}
 \item Investigar métodos de modelagem e comunicação de agentes apropriados para as tarefas envolvidas na orquestração de um ambiente de CN;
 \item Modelar o funcionamento de uma plataforma de SMA que suporte a gerência autonoma de um ambiente de CN;
 \item Avaliar as plataformas de SMAs existentes que podem ser utilizadas para estender uma ferramenta de orquestração de CN;
 \item Implementar funções para criação e especificação de agentes para diferentes componentes de um ambiente orquestrado pela ferramenta \hyphenation{Cloud-Stack} CloudStack, visando especificamente os que estão envolvidos no processo de consolidação de recursos;
 \item Avaliar o funcionamento da plataforma.
\end{itemize}
