Nesta seção serão apresentados os objetivos do trabalho, de forma a esclarecer as metas estabelecidas durante sua elaboração.

\subsection{Objetivo Geral}

Este trabalho busca incrementar uma ferramenta de orquestração de nuvem para que esta use técnologias de sistemas multiagentes para o gerenciamento de ambientes de CN, visando uma melhor gerência dos recursos orquestrados.

\subsection{Objetivos Específicos}

Os objetivos específicos deste trabalho são:

\begin{itemize}
 \item Investigar métodos de modelagem e comunicação de agentes apropriados para as tarefas envolvidas na orquestração de um ambiente de CN;
 \item Elaborar um modelo de SMA baseado em um dos métodos estudados para o suporte de gerenciamento de ambientes de CN;
 \item Avaliar as plataformas de SMAs existentes que melhor se encaixam neste escopo;
 \item Implementar o modelo proposto com base na escolha da plataforma SMA;
 \item Avaliar o funcionamento da ferramenta final.
\end{itemize}
