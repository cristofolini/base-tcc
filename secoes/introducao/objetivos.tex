Nesta seção serão apresentados os objetivos do trabalho, de forma a esclarecer as metas estabelecidas durante sua elaboração.

\subsection{Objetivo Geral}

Este trabalho busca usar de uma plataforma multiagente alocado em uma ferramenta de orquestração de nuvem para suportar o desenvolvimento de agentes de gerenciamento de ambientes de CN, visando uma melhor gerência de recursos.

\subsection{Objetivos Específicos}

Os objetivos específicos deste trabalho são:

\begin{itemize}
 \item Investigar métodos de modelagem e comunicação de agentes apropriados para as tarefas envolvidas na orquestração de um ambiente de CN;
 \item Avaliar as plataformas de SMAs existentes que podem ser utilizadas para extender uma ferramenta de orquestração de CN;
 \item Alterar o funcionamento de uma plataforma de SMA existente para que suporte a gerência autônoma de um ambiente de CN;
 \item Implementar funções para criação e especificação de agentes para diferentes componentes de um ambiente orquestrado pela ferramenta \hyphenation{Cloud-Stack} CloudStack, visando especificamente os que estão envolvidos no processo de consolidação de recursos;
 \item Avaliar o funcionamento da ferramenta final.
\end{itemize}
