Tendo em vista a forma na qual diferentes autores expressam a aptitude de um SMA para gerenciar recursos de forma autônoma, dadas as complicações impostas por um ambiente de CN, este trabalho propõe a expansão da solução implementada por \citeonline{gabriel} com o objetivo de integrar seu agente de consolidação a uma plataforma multiagente, permitindo assim o desenvolvimento de outros agentes que possam atuar na gerência dos recursos do ambiente tanto quanto a cooperação entre os agentes.

A plataforma multiagente será implementada através da ferramenta JADE, citada anteriormente. O JADE permite instanciar e gerenciar agentes reativos e age como uma central de comando para os agentes, sendo toda a instanciação, comunicação e gerencia das atividades dos agentes feita pela plataforma. Os agentes serão desenvolvidos utilizando a linguagem Java para descrever suas ações, planos e possíveis intenções.

A plataforma será instanciada em uma máquina virtual dedicada a funções de sistema dentro de um ambiente de nuvem orquestrado pelo CloudStack. A instalação da ferramenta nessa VM deve ocorrer de forma automática, sem ser necessáia a interferência de um administrador. Para isso, serão utilizadas funções implementadas na plataforma previamente para instanciar uma máquina virtual contendo uma \emph{Java Virtual Machine}, necessária para o funcionamento da plataforma.
